How to use Unicode Explorer

Unicode Explorer
  1. Open the Unicode Explorer from the button above. You will see plenty of Unicode values.
  2. Scroll to find values ranging from E000 to EFFF. Click the value you want to copy.
  3. Click copy on the value page. It should say "Copied to clipboard".
  4. Open Minecraft, join CraftersMC, and paste the Unicode value into your Book and Quill.

Direct Copy Unicode Values

Irrl's Glyph

Click the button above to access glyphs for copying. After loading, click on the glyph you want, and it will be copied to your clipboard. Paste it in your Book and Quill in CraftersMC.

Notes

Not all Unicode values will work. E400-407 are store ranks, whereas E420-425 are staff ranks. If nothing appears, try a different value.

Pasting "E400" etc will NOT WORK. You MUST copy a Unicode value using the websites given above.
